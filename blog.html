Important FAQ

What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

Because it can use methods like `document.getElementById()` or `document.querySelector()` to select elements from the DOM.

What is event delegation, and why is it useful?

Event delegation is a technique that allows someone to handle events at a higher level in the DOM structure rather than directly attaching listeners to individual elements.

How do you manipulate an element's attributes and styles using the DOM?

I can use the `setAttribute` method to change attributes and modify the `style` property to change the inline styles of an element. For example: `element.setAttribute('src', 'new-image.jpg')` or `element.style.backgroundColor = 'blue'`.
